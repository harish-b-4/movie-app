# Movie App – React + Vite + Tailwind CSS

A modern and responsive Movie App built using React, Vite, Tailwind CSS, and a public Movie API.
This project allows users to explore trending movies, view details, and manage their own personalized watchlist.

# Features

 Fast and optimized development setup with React + Vite.

 Responsive UI designed with Tailwind CSS.

 Browse movies fetched dynamically from an external Movie API.

 Add to Watchlist – Save your favorite movies with one click.

 Genre Filter – Filter movies in your watchlist by selecting genres.

 Fully optimized for mobile, tablet, and desktop.

# Pages

Home Page: Displays trending/popular movies from the API

Watchlist Page: Shows all saved movies with a smooth genre-filtering experience

# Tech Stack

React
Vite
Tailwind CSS
Movie API (for fetching movie data)
